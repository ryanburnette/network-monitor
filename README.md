# [network-monitor][1]

A network monitor.

Early work in progress.

Let's keep it simple.

## Goals

- Monitor by
  - ping
  - option to check if port is open (carefully)
  - option to check http
  - option to check https
  - check certificate/expiration
  - option to check smb
  - what other services do we need?
